# RateWatch – Currency Exchange Dashboard

RateWatch is a Python-based dashboard that tracks **live currency exchange rates**, visualizes **historical trends**, and provides **short-term predictions** with actionable insights (Buy/Wait suggestions).  
This project was developed as part of an academic initiative to combine **data visualization** and **predictive analytics** for financial decision support.

---

## Features

- **Live Exchange Rate Tracking:** Fetches real-time currency exchange data from APIs.  
- **Historical Data Visualization:** Displays time-series charts using matplotlib.  
- **Trend Prediction:** Applies **Linear Regression** to forecast short-term movements.  
- **Buy/Wait Recommendation:** Offers simple trading guidance based on predicted trends.  
- **Data Processing:** Uses pandas and NumPy for efficient data cleaning and analysis.  

---

| Category | Tools / Libraries |
